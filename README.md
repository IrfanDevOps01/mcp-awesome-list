## 🔑 Quick Setup Example

Most MCP servers can be configured in your IDE with a simple JSON config:

```json
{
  "mcpServers": {
    "atlassian": {
      "url": "https://mcp.atlassian.com/v1/sse"
    },
    "stripe": {
      "url": "https://mcp.stripe.com"
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": { "GITHUB_TOKEN": "<your-token>" }
    }
  }
}
```

## 💡 Pro Tips

- Always prefer **official vendor-hosted MCP servers** (like Atlassian, Stripe, Cloudflare) — zero local setup, OAuth auth, always up-to-date
- Check the vendor's official docs first before using a third-party alternative
- The MCP ecosystem is growing **FAST** — new servers are being added weekly
