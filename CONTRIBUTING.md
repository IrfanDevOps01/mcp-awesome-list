# 🤝 Contributing to Awesome MCP Servers List

Thanks for your interest in contributing! This list grows with community help.

## How to Contribute

### Adding a New MCP Server

1. Fork this repo
2. Edit `README.md`
3. Add your entry to the correct category table
4. Submit a PR

### Entry Format

```
| # | Name | Use Case | Type | [GitHub](url) or [Hosted](url) |
```

- **#**: Next number in the category
- **Name**: MCP server name
- **Use Case**: Brief description (keep it short)
- **Type**: `Official (Vendor)` or `Third-party`
- **URL**: Use `[GitHub](url)` for repos or `[Hosted](url)` for cloud endpoints

### Rules

- ✅ Verify the URL works before submitting
- ✅ Place the entry in the correct category
- ✅ Keep descriptions concise
- ✅ Mark Official vs Third-party correctly
- ❌ No duplicate entries
- ❌ No broken/404 links
- ❌ No paid-only servers without a free tier

### Reporting Broken Links

Open an issue with:
- The broken link
- The MCP server name
- A replacement link (if you have one)

### Suggesting New Categories

Open an issue with the proposed category name and at least 2 MCP servers that fit.
